# Python Calculator

This project is a simple command-line calculator built using Python.

It was developed as part of the **Problem Solving and Programming** course.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## How to Run

Clone the repository:

git clone https://github.com/Abhi123-avi/python-calculator.git

Go to the project folder:

cd python-calculator

Run the program:

python calculator.py

## Example

Welcome to Calculator

Choose one operation:
1. Add
2. Subtract
3. Multiply
4. Divide
